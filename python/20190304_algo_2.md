## 알고리즘 평가의 기준

시간, 공간(메모리)



## 시간 복잡도

데이터가 많아질수록 걸리는 시간이 얼마나 급격히 증가하는가



## 점근 표기법(Big-O)

절대적인 시간이 아닌 성장성이 중요

|             | 선형 탐색 | 이진 탐색 |
| ----------- | --------- | --------- |
| 최고의 경우 | O(1)      | O(1)      |
| 최악의 경우 | O(n)      | O(lg n)   |

시간 복잡도는 대게 다음 중 하나이다.

- O(1)
- O(lgn)
- O(n)
- O(nlgn)
- O(n^2)
- O(n^3)
- O(n4)
- O(2n)
- O(n!)
